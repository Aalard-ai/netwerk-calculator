NETWERK CALCULATOR - GEWENSTE FUNCTIONALITEIT
---------------------------------------------
Doel: Uitgangspunten voor de bandbreedte tool.

1. Invoer: Bestandsgrootte (MB, GB, TB)
2. Invoer: Netwerksnelheid (Mbps, Gbps, 10Gbps)
3. Berekening: Gebaseerd op decimale eenheden (1000).
4. Overhead: Inclusief 10% TCP/IP correctie voor real-world scenario's.

Code-instructie:
Gebruik de formule: Tijd = Grootte / Snelheid.
Voor real-world: Tijd * 1.11 (voor 90% efficiency).
